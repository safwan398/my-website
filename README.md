# رحلة نُجْم في المغرب

لعبة منصات 2D مستوحاة من ألعاب ماريو لكن بطابع مغربي.

## طريقة التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## الإضافات

- موسيقى خلفية خفيفة مولدة داخل المتصفح
- مؤثرات للقفز والجمع والخسارة والفوز
- زر كتم وتشغيل للصوت

## التحكم

- الأسهم أو `A / D` للحركة يمين ويسار
- `W` أو `Space` للقفز
- `S` أو السهم السفلي للانحناء

## المراحل

1. المدينة الزرقاء - مصاعد متحركة بين الأسطح
2. سوق مراكش - وسائد ارتداد وعربات متحركة
3. جبال الأطلس - رياح تدفع اللاعب
4. صحراء مرزوكة - سجاد طائر ورمال خطرة
5. ميناء الدار البيضاء - مفتاح وجسر أخير مغلق
